Embedding

Add a snippet to any website

1

Click the </> Embed button on any widget below to get a ready-to-paste iframe for that snippet.

2

Paste the single <iframe> tag anywhere in your HTML:

<iframe
  src="https://USERNAME.github.io/REPO/viewer.html?src=https://USERNAME.github.io/REPO/snippets/hello.json"
  width="100%"
  height="540"
  style="border:none;border-radius:8px;"
  loading="lazy"
></iframe>
3

That's it — no script tags, no dependencies on the host page. Each snippet is fully self-contained.

Powered by Skulpt and CodeMirror 6. No server required.
